Customizer Remove All Parts

Removes all traces of the WordPress Customizer from your admin.

View on GitHub View on WordPress.org Download .zip

About

WordPress Customizer Remove All Parts plugin for WordPress allows you to get the Customizer out of your WordPress install. It prevents any Customizer scripts from loading and removes all links and buttons to Customizer in the dashboard and admin toolbar.
